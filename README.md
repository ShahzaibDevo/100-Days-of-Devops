

# 🚀 100 Days of DevOps Challenge

Welcome to my **100 Days of DevOps** journey! This repository serves as a comprehensive log of my technical growth, documenting daily tasks ranging from Linux administration and Version Control to Containerization, CI/CD, and Infrastructure as Code (IaC).

## 🛠️ Tech Stack & Tools

* **Linux:** User management, Networking, Security (SELinux/IPTables).
* **Version Control:** Git & GitHub (Branching, Merging, Rebasing).
* **Containerization:** Docker & Docker Compose.
* **Orchestration:** Kubernetes (Pods, Deployments, PV/PVC, Secrets).
* **CI/CD:** Jenkins (Freestyle jobs, Pipelines, Slave Nodes).
* **Automation:** Ansible (Playbooks, Modules, Inventory).
* **Infrastructure as Code:** Terraform (AWS, VPC, EC2, IAM).
